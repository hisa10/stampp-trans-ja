Editing the Accident Description

Accident Description

Here you can shortly descripe the accident. How it happens and more detailed infos about it. You can also edit the text via the Toolbox and change the layout of the text. For example you can change the font or make the text bold and more.

Accident Date and Time

Here you can set the time and date for the accident.

Accident Location

You can give here additional and extended information about the accident location.

Picture List

Here you can add pictures of the accident by clicking the add button. You can also delete pictures. If you want to see the picture in a big view you have to click the button

Company Information

It is possible to give addition information about the company which was responsible for the accident.
