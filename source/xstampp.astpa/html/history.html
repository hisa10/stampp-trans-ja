2.0.0:

Since this version A-STPA is now part of the eXtensible STAMP Platform (XSTAMPP) it is now provided as fully integrated plug-in.
Due to this A-STPA has now got a whole new appearance which should enable the user to use it in a far more flexible way. Features:
  • Possibility of managing multiple projects at the same time thanks to the XSTAMPP project explorer
  • toolbars now provided in the main toolbar of the workbench
  • Renaming Projects directly in the project explorer
