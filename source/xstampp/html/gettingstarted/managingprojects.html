Managing a Project

Creating a Project

There are two ways to create a new project. If you are at the Welcome Screen, click on and a new project will be created. Otherwise go to File in the menu bar and click on Create New Project.

Saving a Project

Changes on an existing project can saved by navigating to the menu bar, clicking on File -> SaveProject. When saving for the first time, you will be asked to enter a file destination and a file name.

Exporting a Project

The PDF export is used to create a document that is easy to share. Go to File in the menu bar and click Export As.. . A small window will appear, that can be used to change the export parameters.
  • Company The company name can be entered in order to be shown on the front page of the PDF export file.
  • Logo A company logo or other random image can be defined to be shown on the front page of the PDF export file.
  • Background and Font Color Colors can be changed for a custom page design.
  • Destination The desired destination of the PDF file can be set here.
